# **Exploratory-Healthcare-Patient-Data-Analysis-and-Visualization-Using-Python**
The project helps to generated meaningful statistical summaries to better understand data behavior. Created effective visualizations to present insights in an easy-to-understand manner

# **Project Overview**

- This project performs Exploratory Data Analysis (EDA) on a healthcare dataset to uncover insights about patient demographics, hospital stay duration, treatment patterns, and medical billing.

- The goal of the project is to transform raw healthcare data into meaningful insights that can support data-driven decision making in healthcare management.

- The project demonstrates the complete data analytics workflow, including data cleaning, exploratory analysis, visualization, and interpretation of results.

# **Project Objectives**

Healthcare organizations generate large volumes of patient data, but much of this data remains underutilized due to missing values, inconsistency, and lack of proper analysis. There is a need to clean, analyze, and visualize healthcare data to uncover meaningful insights that can support better patient care and informed decision-making. This project addresses the challenge of transforming raw healthcare data into actionable insights using Python-based data analysis techniques

- Understand the structure and characteristics of the healthcare dataset

- Clean and preprocess raw data for accurate analysis

- Perform Univariate, Bivariate, and Multivariate analysis

- Identify patterns and relationships between variables

- Detect and handle missing values, duplicates, skewness, and outliers

- Generate business insights from healthcare data

- Provide recommendations for improving healthcare operations

# **Key Insights**

The statistical summary provides key metrics such as mean, median, and standard deviation, helping understand the general behavior of the data. Grouping by gender helps identify differences in healthcare costs and hospital stay durations. Frequency counts of treatment types highlight which treatments are most common, helping hospitals allocate resources effectively. The scatter plot shows that billing amount increases as hospital stay duration increases, indicating that longer stays drive higher healthcare costs. The bar plot reveals that some treatment types require longer hospitalization, which explains why certain procedures result in higher expenses. These insights help healthcare managers identify cost drivers and operational bottlenecks. The regression line suggests that billing costs increase proportionally with hospital stay duration. Hospitals can use this trend to forecast patient expenses and resource usage. Predictive insights help in budget planning and insurance cost estimation.Hospitals should monitor treatments that lead to longer hospital stays, as they significantly increase operational costs. Improving discharge planning and post-treatment care can reduce extended hospitalization. High-cost treatments should be analyzed for resource optimization and cost control strategies. Healthcare administrators can use patient trend data to optimize staffing, bed allocation, and medical resources. Data-driven planning will help improve patient care quality while controlling expenses.
